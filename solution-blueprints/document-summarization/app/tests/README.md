# Automated Testing Guide

## Environment Setup
Make sure your `.env` variables or shell exports are active.

## Launch Instructions
Use the provided shell script to run validation tests (specifically for ROCm/vLLM setups):

```bash
chmod +x test_compose.sh
./test_compose.sh
```
